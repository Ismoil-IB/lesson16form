Form это главный отрибут для создания формы.
Значения action- получатель данных формы.
Method - это метод передачи данных.
Значения сврйсива Method get - значение по умолчанию, данные передаются в адресной строке,
Method post - данные передаются скрытно enctype - тип данных.
Свойство enctype - это тип данных. значения (для передачи фалов, указываем multipart/form-data)

Тег input это поле для ввода данных
Value свойство которое может заполнить поля для ввода
Name свойство котором называют тег input
Required свойство которое делает обязательным поле для ввода
Maxlength максимальое значения для ввода данных в поле input
Placeholder это свойство подписывает input указыват что нужно в вести в поле для ввода
Autocomplete отключает или включает возможность подсказыват браузеру
Checked зарания выберает один из значение
Disabled отключает исползования тега
Tabindex="" задаёт последовательность табуляции

Значения свойства type="number" ввод чисел

Значения свойства type="tel" ввод номера телефона

Значения свойства type="email" ввод электронной почты

Значения свойства type="password" ввод пароля

Значения свойства type="file" загруска файла.
Multiple позволяет выбрать несколько файлов.
Accept=".jpg, .png" позволяет выбрать файлы определённого типа

Значения свойства type="date" календарь
type="datetime-local" календарь + время

Значения свойства type="checkbox" выбор И-множественный выбор (чекбокс). Value передаёт значения yes, no или on, off

Select Выподаюший список
Selected зарания показывает какой пункт можно выбрать
Multiple позволяет выбрать несколько вариантов выпадаюшего списка

Типы кнопки button:
type="button" обычная кнопка
type="reset" спросит заполненые поля
